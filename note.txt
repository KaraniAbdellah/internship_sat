# Design AI Agent System for Generate Marketing Offres
- Goal: >= 70% Automation, Satisfaction > 4.5/5, Response 95% < 2.5s
- Scope: Generating, Scoring, Validation and Optimisation
- Workflow:
    - Client Input: "I want to offer discount to tech professionals" + CLIENT DATA
    - Agent 1 (Generation): Creates offer → "30% off Pro plan"
    - Agent 2 (Scoring): Rates offer → "Score: 8/10"
    - Agent 3 (Validation): Checks if valid → "Approved ✓" --> Based on Rules (max discount 50%, ...)
    - Agent 4 (Optimization): Improves it → "Add urgency: 48hr deadline"
    - Show Result: Client sees final optimized offer
- Rules:
    - Generate Marketing Offre Based on Client Data
    - Score The Offre (0<=S<=100) and Predict churn 
    - Validate The Offre (Human Validation)
    - Optimise The Offre (Suggest Optimisation on Some Parts)
- Backend and Database
    - Data Storage to Relation Database or Vector DB

    - Agent-to-Agent Communication
